# Zenith Clicker: Shimmering Waves

    Welcome to Shimmering Waves, a small mod of Zenith Clicker made by FMichael.  

> Almost all art and sounds are from [TETR.IO](https://tetr.io), a modern yet familiar online stacker game by osk with amazing graphics and music.

**NOTICE**: This mod requires knowledge from base Zenith Clicker. PC is *highly recommended*. Web and Mobile currently are not supported.

Powered by LÖVE & Zenitha & Lua ~~instead of slow web engine~~
